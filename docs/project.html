Github Project

For this project, I analyzed CDC heart disease mortality data among U.S. adults aged 35 years and older from 2018 to 2020. I built a reproducible R workflow to import, clean, and recode the dataset, generate descriptive statistics, create visualizations of mortality patterns by demographic and geographic factors, and fit regression models to identify factors associated with heart disease mortality rates. The project also included an automated reporting pipeline that renders the final HTML report, with outputs organized into separate folders for cleaned data, figures, tables, and the final report. To improve reproducibility, I used renv, a Makefile, and Docker so the full analysis can be rebuilt in a consistent computing environment.

Thesis Interactive Dashboard

For this project, I developed an interactive thesis dashboard to visualize cardiometabolic indicators among females with classic 46,XX congenital adrenal hyperplasia and matched female referents. Using R Markdown, ggplot2, dplyr, and Plotly, I created interactive boxplots and age-related trend plots for laboratory and blood pressure measures, including glucose, hematologic, lipid, and blood pressure outcomes. Because the underlying clinical data are confidential, the raw dataset is not publicly available; however, the dashboard presents de-identified, aggregated visual summaries that allow users to explore group-level patterns. This project was designed as a visual companion to my thesis analysis and demonstrates my ability to translate longitudinal clinical data into an accessible, interactive web-based format.
