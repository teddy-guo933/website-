# Zihan (Teddy) Guo's Personal Website

This repository contains the source files for my personal academic and professional website. The website highlights my background in public health and epidemiology, research interests, technical skills, selected projects, curriculum vitae, and resume.

## Website

The website is available at:

https://teddy-guo933.github.io/website-/

## Website Sections

- **Home:** Brief introduction, education background, research interests, and personal interests
- **About:** Research interests, current job interests, technical skills, laboratory skills, and professional strengths
- **Projects:** Selected GitHub and R projects, including a heart disease mortality analysis project and an interactive thesis dashboard
- **Curriculum Vitae:** Academic and professional CV
- **Resume:** Current resume

## Tools Used

This website was built using:

- R Markdown
- HTML/CSS
- Font Awesome icons
- GitHub Pages

Some projects shown on this website use confidential clinical or research data. Raw datasets are not publicly available, but de-identified and aggregated summaries or visualizations may be shown when appropriate.
